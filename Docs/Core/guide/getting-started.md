# Getting Started

Welcome to Ishmael PHP. This guide helps you run the SkeletonApp and explore the framework.

## Prerequisites
- PHP 8.2+
- Composer (for development convenience)
- Optional: Laravel Herd for local `.test` domains

## Quick start with SkeletonApp
1. Copy `.env` or let the framework generate one on first run.
2. Serve the `SkeletonApp/public` directory with your web server or PHP's built-in server.
3. Visit the app in your browser (for Herd: `http://ishmaelphp.test`).

## Next steps
- Read the Routing guide.
- Explore Modules under `SkeletonApp/Modules`.
